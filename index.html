MyPantry

This fullstack app is created as a virtual pantry, it can keep track of pantry, add items, check items in pantry, remove items, and check expiration date.

The project started as an idea, instead trying so hard to remember how many tomatoes are sitting on the kitchen counter, an app should be able to do that faster and more accurately, that was the inception of MyPantry

The app was built in React(Frontend) and Express(Backend). Database was built using PostgreSQL.

LyriSongs

An app that conveniently lets you search for a Band/Musician, search for a song and get the lyrics and a YouTubeVideo for that song all in one platform. Start by searching for a Band/Musician, the app will display a short bio and some general information, then any particular song can be searched and the app wil retrieve a YouTube video of that song, and under that will display the lyrics.

The app was built using HTML, CSS, JavaScript and JQuery. Three different API's are used in this app, YouTube API, The ADB API and Lyrics OVH API.
